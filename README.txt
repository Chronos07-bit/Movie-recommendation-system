# Content-Based Movie Recommendation System

This project recommends movies similar to a given movie based on genre similarity using content-based filtering.

## Features
- TF-IDF Vectorization of movie genres
- Cosine similarity for recommendations
- Genre filter for refined suggestions
- Streamlit interface for interactive use

## How to Run
1. Install required packages:
   pip install streamlit pandas scikit-learn
2. Place 'movies.csv' and 'app.py' in the same folder
3. Run the app:
   streamlit run app.py
